## üåü What is OnchainLab?

OnchainLab is a comprehensive, production-ready platform that empowers developers and creators to build, deploy, and manage Web3 applications effortlessly. It provides a complete suite of smart contract factories, development tools, and utilities that eliminate the complexity of blockchain development while maintaining enterprise-grade security and reliability.

Think of OnchainLab as your all-in-one Web3 development studio - everything you need to launch tokens, NFTs, DeFi protocols, and smart contract applications is right at your fingertips.

### Step 1: Connect Your Wallet
Connect your Web3 wallet (MetaMask, WalletConnect, etc.) to Mantle Sepolia Testnet. The platform will guide you through adding the network if needed.

Click deploy and watch your smart contracts come to life. All complex blockchain operations are handled automatically.

### Step 5: Interact with Your Creation
Your deployed contracts are immediately ready to use. View them on Mantlescan, interact with them, or integrate them into your applications.

## üö¶ Getting Started

### For Developers

1. **Clone the Repository**
   ```bash
   git clone https://github.com/Abidoyesimze/OnchainLab.git
   cd OnchainLab
   ```

2. **Install Dependencies**
   ```bash
   cd frontend
   npm install
   ```

3. **Configure Environment**
   ```bash
   cp .env.example .env.local
   # Add your configuration
   ```

4. **Run Development Server**
   ```bash
   npm run dev
   ```

5. **Connect Wallet**
   - Add Mantle Sepolia Testnet to your wallet
   - Chain ID: 5003
   - RPC URL: `https://rpc.sepolia.mantle.xyz`

6. **Start Building**
   - Navigate to the tool you need
   - Configure your parameters
   - Deploy your contracts

### For Non-Technical Users

1. **Visit the Platform**
   - Open OnchainLab in your browser

2. **Connect Your Wallet**
   - Click "Connect Wallet"
   - Follow the prompts to add Mantle Sepolia

3. **Choose Your Tool**
   - Browse available tools
   - Select what you need

4. **Fill in the Forms**
   - Use our guided interface
   - All fields have helpful descriptions

5. **Deploy**
   - Click deploy
   - Confirm the transaction
   - View your contract on Mantlescan
